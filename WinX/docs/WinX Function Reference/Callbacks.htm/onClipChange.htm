onClipChange (callback)

onClipChange ()

This callback is invoked when the clipboard data changes

Parameter Description

Returns

0

Remarks

This is a callback function. You must implement it yourself. To register this callback, use the WinXRegOnClipChange function.

Examples

FUNCTION onClipChange ()

IF WinXClip_IsString() THEN PRINT "Clipboard now contains a string"

END FUNCTION
