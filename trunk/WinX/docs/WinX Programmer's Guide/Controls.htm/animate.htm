Animation Controls

Animation controls are a simple way to play an animation. They are usually used to indicate that an application is busy. Animation controls play uncompressed AVI files without sound only. Under Windows XP and higher, if you use the XP style controls and the AVI file has an alpha channel, fully transparrent parts of the animation will be rendered as transparent.
