WinXRegOnColumnClick

WinXRegOnColumnClick (hWnd, FUNCADDR onColumnClick)

Registers an onColumnClick callback function

Parameter Description
hWnd The handle to the window to register the callback for
FUNCADDR onColumnClick The address of the onColumnClick callback function to register

Returns

$$TRUE on success or $$FALSE on fail

Remarks

This callback is invoked when the user clicks a column header in a list view control.

Examples

' Register an onColumnClick callback function
WinXRegOnColumnClick (#hMain, &onColumnClick())
