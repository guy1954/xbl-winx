WinXToolbar_AddSeperator

WinXToolbar_AddSeperator (hToolbar)

Adds a seperator to a toolbar

Parameter Description
hToolbar The toolbar to add the seperator to

Returns

$$TRUE on success or $$FALSE on fail

Remarks

The toolbar displays buttons in the order they are added.

Examples

'add a seperator to a toolbar
